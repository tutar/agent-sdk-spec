# Durable Memory Types

`memory-types/` 定义 durable memory 的标准 taxonomy。

这些类型在默认 local mapping 中通常由 `auto-memory` prompt/runtime 引入，但语义层属于 durable memory type system，而不是某个具体 runtime 私有实现。
